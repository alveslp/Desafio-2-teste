🚀 Desafio Prático: Análise de Engajamento de Streaming

Este repositório contém a resolução do desafio prático de modelagem, tratamento e cruzamento de dados em escala utilizando o Google BigQuery e SQL. O objetivo principal foi analisar métricas de engajamento de usuários (minutos assistidos) segmentados por país, aplicando conceitos avançados de junção de tabelas (INNER JOIN), agrupamento (GROUP BY) e filtros.

🛠️ Tecnologias Utilizadas
Google Cloud Platform (GCP)

Google BigQuery

SQL (Standard SQL)

Markdown (Documentação do Repositório)
